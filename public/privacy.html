Política de Privacidade

Versão demo

Esta página é um exemplo para fins de demonstração. Ajuste para sua realidade (LGPD, DPO, bases legais, prazos de retenção etc.).

1. Quem somos
Este site demonstra um sistema de mensagens via WhatsApp Cloud API (Meta) para atendimento automatizado.

2. Quais dados coletamos
Podemos processar dados fornecidos via WhatsApp, como número de telefone, nome de perfil e conteúdo da mensagem, além de logs técnicos (data/hora).

3. Finalidade
Atendimento ao cliente, resposta a solicitações e demonstração de envio/recebimento de mensagens via WhatsApp.

4. Compartilhamento
Mensagens são transmitidas pela infraestrutura do WhatsApp/Meta, conforme as políticas e termos do WhatsApp.

5. Retenção
Nesta demo, os registros são mantidos temporariamente em memória e podem ser perdidos ao reiniciar o servidor.

6. Segurança
Recomendamos uso de HTTPS, controle de acesso, segregação por cliente e auditoria para ambiente de produção.

7. Contato
Para solicitações relacionadas à privacidade, entre em contato: seu-email@dominio.com.
