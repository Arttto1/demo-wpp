Criação de modelo (template)

Vídeo separado

Para a permissão whatsapp_business_management, a Meta pede um vídeo mostrando criação de um modelo de mensagem. O jeito mais simples/aceito é criar pelo WhatsApp Manager.

Checklist do vídeo

  • Abra o painel do WhatsApp Manager (na conta correta).
  • Vá em Message templates (Modelos de mensagem).
  • Crie um template simples (ex.: confirmação de agendamento).
  • Mostre o template criado e o status (enviado para aprovação / aprovado).

Dica
Se você quiser, eu posso adicionar um endpoint via Graph API para criar template programaticamente. Mas para passar na análise, normalmente o vídeo pelo Manager é o caminho com menos risco.
